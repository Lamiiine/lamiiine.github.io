Save time with these Linux Shell commands

Jul 19, 2021

Here I put the list of Linux shell commands I use frequently and I find efficient and good to know

sudo !! (or sudo bang bang):

A lot of the time, when I copy a command, paste in shell and quickly press Enter, I realize I forgot to write sudo first. No worries, sudo !! rerun the last command with sudo prepended to it.

Cut a word at a time backwards:

Sometimes you want to remove a word backwrods in your command.Ctrl + w will do the work for you.

$cut -d: -f1 < /etc/passwd | sort | xargs

Ctrl + w right after xargs gives:

$cut -d: -f1 < /etc/passwd | sort |

Type Ctrl + w again gives:

$cut -d: -f1 < /etc/passwd | sort

Cut to the end of a line:

If you are trying to eliminate/Cut an argument in your command, simply type Ctrl + k.

$cut -d: -f1 < /etc/passwd | sort | xargs

Ctrl + k right after sort gives:

$cut -d: -f1 < /etc/passwd | sort

Cut to the beginning of the line:

This is the inverse of cutting to the end of the line. That is if you want to cut a command to the beginning of the line you simply type Ctrl + u

$cut -d: -f1 < /etc/passwd | sort  | xargs

Ctrl + u before sort gives:

$ sort  | xargs

Paste back the removed/cut line:

If you are trying to paste back the line you removed whether to the beginning of the line or to the end of it, you simply type Ctrl + y which returns the last thing you cut.

$cut -d: -f1 < /etc/passwd | sort

Ctrl+ y right after sort gives:

$cut -d: -f1 < /etc/passwd | sort  | xargs
